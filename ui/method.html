System Help

Usage

NEPRE-F Prediction

1. Choose a PDB file(Use .pdb file format).

2. Set the energymatrix or use default setting(cutoff=6).

3. Click "GO" and start to predict.

NEPRE-R Prediction

1. Choose a PDB file(Use .pdb file format).

2. Choose the radius file or use default(System Recommended).

3. Choose the energy matrix.

4. Click "GO" and start to prdict.

Energy Matrix Generate

Both NEPRE-F and NEPRE-R have their own energy matrix. In addition to the default values provided by this system, we also allow users to calculate their own. Users can generate energy matrix by following steps:

1. Choose which algorithm your energy matrix are used.

2. Set the PDB dataset path.

3. Input the radius file(NEPRE-R) or set the cutoff(NEPRE-F)

4. Click "GO" and start to generate.

The energy matrix will be save as .npy format in the workding directory "cache".
